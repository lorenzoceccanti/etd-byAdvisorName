A simple TamperMonkey script restoring the "Search by Advisor" functionality in Unipi's ETD digital archive.

## Instructions
- Install the TamperMonkey extension: [Firefox link](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/), [Chrome Link](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo?pli=1)
- From Extension Toolbar select <i> Dashboard </i>
- Then, on the new window, select <i> Utilities </i>
- At the bottom of the new page, in the section <i> Import from URL </i> insert the following:
  [https://raw.githubusercontent.com/lorenzoceccanti/etd-byAdvisorName/refs/heads/main/etd_by_advisor_name.js](https://raw.githubusercontent.com/lorenzoceccanti/etd-byAdvisorName/refs/heads/main/etd_by_advisor_name.js)
- You should see the search by advisor functionality in the top-right of the webpage. A text field will appear expecting the Last Name of the Advisor to query on.
